# BitquantAutoBot-NTE
## Instalation

1. **Clone The Repositories:**
```bash
git clone https://github.com/VinzSenzoo/BitquantAutoBot && cd BitquantAutoBot
```

2. **Install Dependencies**
```bash
npm install
```

3. **Enter Your Private Key**
```bash
nano pk.txt
```

4. **If You Want use proxy enter your proxy**
```bash
nano proxy.txt
```

5. **Run Bot**
```bash
npm start
```
